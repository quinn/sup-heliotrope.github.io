Hi all,

I've put a slightly modified Roundup installation here for tracking Sup
bugs:

  http://masanjin.net/sup-bugs/

Please use this to submit your bug reports and feature requests. If you
have existing bug reports or feature requests and you don't want to rely
on my faulty memory any more, feel free to resubmit them here.

You should be able to create new issues without having to register.
However, if you want to receive email about an issue when updates
happen, you will have to register. Sorry! On the plus side, registration
is super quick because I've disabled the email confirmation step.

are a couple things I'd like to tweak.
