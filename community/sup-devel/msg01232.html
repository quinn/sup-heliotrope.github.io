Can there be an optional functionality to flush changes to disk every 10-15
minutes, instead of having to press $ every time

Rationale: I'm in a place where power cuts are extremely frequent. I keep
forgetting to flush changes manually requiring me to re-delete and archive my
messages every time I reboot and restart sup.

I run sup off of git 'next' head. A patch which will hard-code this
functionality would be fine as well. Alternatively, is there a command I can
call via a cron job to do this?
